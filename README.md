🏢 The Bureaucracy

"Accuracy is mandatory. Sanity is optional."

The Bureaucracy is a stress-inducing browser game built with React and Tailwind CSS. The goal is simple: fill out infinite government forms correctly before your sanity (or the timer) runs out.

🎮 How to Play

Read the Red Text: Every field has a specific instruction (e.g., "Type 'AGREE' backwards").

Ignore Common Sense: If the form asks for your age, and the instruction says "Enter a negative number," do it.

Watch the Timer: If time runs out, you get a strike.

Don't Lose Your Mind: Incorrect answers lower your Sanity meter.

🛠️ Tech Stack

React (Vite)

Tailwind CSS (Styling)

Lucide React (Icons)

Vercel (Deployment)
